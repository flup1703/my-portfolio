# Widgets

Назначение: переиспользуемые составные блоки интерфейса, которые объединяют несколько UI-компонентов и могут использоваться в разных фичах/страницах.

Пример: `ProjectCard`, `ContactCard` могут стать виджетами в `src/widgets`.

Дальнейшие шаги:
- Перенесите переиспользуемые компоненты из `src/components` в `src/widgets` постепенно.
- Создайте `index.ts` для реэкспорта.
