# Shared

Назначение: общие утилиты, UI-примитивы, темы, конфигурации и типы, используемые по всему приложению.

Структура примера:
- `src/shared/ui` — низкоуровневые UI-примитивы (кнопки, инпуты и т.д.)
- `src/shared/lib` — хелперы/утилиты
- `src/shared/config` — глобальные константы/настройки

Дальнейшие шаги:
- Создайте баррели `index.ts` и постепенно реэкспортируйте существующие файлы из `src/components` и `src/lib`.
