# Shared UI

Назначение: здесь находятся UI-примитивы (кнопки, инпуты, иконки и т.д.), которые не зависят от доменной логики.

Рекомендуемая миграция:
- `src/components/ui/*` -> `src/shared/ui/*`

Пример команды:
```bash
git mv src/components/ui src/shared/ui
```

После перемещения добавьте `src/shared/ui/index.ts` для центрального реэкспорта.
