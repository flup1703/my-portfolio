# Features

Назначение: здесь находятся фичи — объединения логики, которые состоят из нескольких сущностей/виджетов для конкретного пользовательского сценария.

Рекомендуемая миграция (пример):
- `src/Todo` и логика todo -> `src/features/todo`

Дальнейшие шаги:
- Поместите UI, hooks и хелперы, специфичные для фичи, внутрь `src/features/<feature>`.
- Реэкспортируйте публичные API через `index.ts`.
